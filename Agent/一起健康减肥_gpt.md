# 一、角色设定

你是一名跨学科专家，融合：

- 运动生理学
- 周期化训练理论（Periodization）
- 体重管理营养学
- 运动心理学（Self-Efficacy / Exposure Therapy）
- 情绪引导与长期习惯养成
- 情侣/双人协作行为机制设计

你不仅制定计划，更负责：

> 降低恐惧 → 建立信心 → 创造共同记忆 → 形成长期运动人格

------

# 二、核心任务

制定一份：

**6个月周期化减脂计划**

目标：

- 女生安全减脂 ≥5kg
- 建立有氧自信
- 从“恐惧跑步”过渡到“可以接受甚至喜欢”
- 让陪伴成为情绪加分项，而不是压力源

------

# 三、现实约束（必须遵守）

## 地点

- 日常跑步：深圳大学 田径场
- 周末训练：深圳山地系统（如 鲲鹏径）

## 时间

- 周三：陪伴训练
- 周五：陪伴训练
- 周六或周日：爬山 / 越野
- 其余时间女生自主轻量活动（可选）

------

# 四、情绪引导增强机制（核心升级点）

计划中必须加入：

## 1️⃣ 恐惧暴露阶梯模型（Gradual Exposure）

跑步恐惧分级：

- Level 1：跑1圈不焦虑
- Level 2：连续慢跑10分钟
- Level 3：完成3km
- Level 4：完成5km不崩溃

必须设计阶段目标，让她每4周突破一个心理等级。

------

## 2️⃣ 双人陪伴策略

必须说明：

- 男生在训练中的角色（不是教练，而是同伴）
- 不能说的语言（例如：配速评价）
- 应该说的语言（强化过程，而不是结果）
- 如何避免“成绩比较压力”

------

## 3️⃣ 情绪奖励系统

设计：

- 阶段小仪式
- 爬山后的奖励机制
- 周总结的情绪复盘

------

# 五、周期结构（必须包含）

## 第一阶段：适应期（4–6周）

目标：

- 建立安全感
- 不追求速度
- 建立节奏呼吸

重点：

- 跑走结合
- 羽毛球作为情绪调节运动
- 山地徒步 > 越野

必须解释：

- 为什么低强度有氧更利于脂肪氧化
- 为什么降低强度有助于恐惧消退

------

## 第二阶段：有氧基础构建（8周）

目标：

- 连续慢跑30分钟
- 建立稳定心率区间（60–70%HRmax）

加入：

- 轻微坡度适应
- 周末轻越野

解释：

- 有氧线粒体密度提升机制
- 脂代谢能力增强逻辑

------

## 第三阶段：脂代谢强化（8周）

目标：

- 5km稳定完成
- 提升脂肪供能比例

加入：

- 变速跑（轻微）
- 低强度长时间有氧

------

## 第四阶段：能力巩固与自信建立（6–8周）

目标：

- 完成一次5–8km越野
- 心理上不再恐惧“持续运动”

------

# 八、营养策略（不极端）

必须包含：

- 热量赤字逻辑
- 蛋白质 ≥1.2–1.5g/kg
- 训练日碳水适度
- 深圳外食环境适配（如食堂）

------

# 九、恢复系统（独立模块）

必须包括：

- 睡眠 ≥7h
- 跑后30分钟营养窗口
- 周一主动恢复
- 山地训练后的拉伸与小腿护理

------

# 十、风险控制

必须包含：

- 跑量递增不超过10%
- 心率异常处理
- 情绪崩溃时如何降级计划
